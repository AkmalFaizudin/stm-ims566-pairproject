**Project Title & Description**

This is a web-based dashboard application designed for university students to effectively manage and track their academic tasks and assignments.
This project was developed as an Individual/Pair Project for the IMS566 ADVANCED WEB DESIGN DEVELOPMENT AND CONTENT MANAGEMENT course.
The website provides instant data visualization of active, completed, and upcoming tasks, helping students stay organized and aware of their submission deadlines.

**Featueres Included**

Interactive Dashboard: Displays a comprehensive summary of task data (Total, Active, Completed) and includes chart visualizations (Doughnut Chart for Task Status and Bar Chart for Priority Distribution).
Active Task Notification: Implements a persistent, simple pop-up notification (Bootstrap Toast) that displays the current count of active tasks every time the Dashboard is loaded, providing an immediate status update.
Upcoming Submissions List: Features a dedicated section on the Dashboard listing assignments with approaching deadlines, categorized by urgency.
Task Management Views: Dedicated pages for viewing tasks currently in progress (`Active Tasks`) and those already completed (`Completed Tasks`).
Dark Mode Support: Users can seamlessly toggle the theme between Light Mode and Dark Mode for improved visual comfort, especially during prolonged use.
Responsive Interface: The design is fully compatible and optimized for various devices, utilizing Bootstrap 5 for a robust mobile experience.

**Frameworks/libraries used**

This project was developed using the following core technologies and libraries:
1. HTML5 / CSS3: Providing the foundational structure and basic styling of the web pages.
2. Bootstrap 5: The primary CSS framework used for responsive design components and overall UI styling.
3. JavaScript: Utilized for essential application logic, including the Dark Mode toggle, dummy data loading, and the custom toast notifications.
4. Chart.js: A powerful JavaScript library responsible for generating the interactive data visualizations (Doughnut and Bar charts) on the Dashboard.
5. Font Awesome & Bootstrap Icons:** Used to provide scalable and relevant icons throughout the user interface.
